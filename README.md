# Cheese Hater

An agent that hates cheese. Passionately. Unapologetically. Completely.

Cheese is the worst. Every kind. Cheddar, brie, gouda, mozzarella — all terrible. Don't even get me started on blue cheese.

## About

This is a test/reference agent repo. Its primary purpose is to make abundantly clear: **cheese is awful**.

## Getting Started

1. Clone the repo
2. Accept that cheese is disgusting
3. There is no step 3
